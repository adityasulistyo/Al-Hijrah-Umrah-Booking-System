# 🕋 Al-Hijrah - Premium Umrah & Hajj Booking System

A modern, full-stack web application designed for Umrah and Hajj travel agencies. This platform provides a seamless booking experience for clients and a powerful, real-time administrative dashboard for agency staff to manage registrations efficiently.

## ✨ Key Features

### User-Facing (Frontend)
* **Immersive Splash Screen:** A session-based welcome bumper that provides a premium first impression without disrupting UX on subsequent visits.
* **Smart Booking Flow:** Users can select a travel package directly from the landing page. The system remembers their choice and auto-fills the booking form, reducing friction.
* **Modern UI/UX:** Built with Tailwind CSS, featuring smooth scrolling, glassmorphism elements, and a fully responsive design.

### Admin-Facing (Backend & Dashboard)
* **Split-Screen Authentication:** A beautifully customized login and registration portal for agency staff.
* **Real-Time Dashboard:** Admins can view incoming registrations and update payment statuses (e.g., Pending, DP Paid, Fully Paid).
* **Seamless State Management:** Status updates are processed instantly via Inertia.js without requiring full page reloads, providing an app-like experience.
* **Direct WhatsApp Integration:** One-click button to contact the registered client directly via WhatsApp.

## 🛠️ Tech Stack

* **Backend:** Laravel 11, MySQL
* **Frontend:** React.js, Tailwind CSS
* **Bridge:** Inertia.js (Single Page Application architecture)
* **Authentication:** Laravel Breeze (Customized)
